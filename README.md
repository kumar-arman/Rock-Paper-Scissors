# Rock-Paper-Scissors
Rock Paper Scissors game using Python with an adaptive strategy that defeats multiple bots with over 60% win rate, built as part of the freeCodeCamp Scientific Computing with Python project.

Rock Paper Scissors is a Python-based project developed for the freeCodeCamp Scientific Computing with Python certification.
The program implements an intelligent player() function that adapts to different opponent strategies and consistently achieves a win rate of at least 60% against all provided bots.

Key features:

Adaptive strategy using opponent history

Defeats all four bots (Quincy, Abbey, Kris, Mrugesh)

Follows freeCodeCamp project constraints

Fully testable using provided test module
